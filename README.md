# Hypernex.Godot

The Hypernex Godot Client

## Status

This client is very Work-in-Progress.

### Features

- Login, Home, Worlds and Instance UI
- Networking
- PC/Flatscreen Movement
- Proximity Voice Chat
- Proximity Text To Speech
- GIF/Video playback in UI

### In Progress

- Godot Editor CCK

### Planned

- Avatars
- VR
- In-Game CCK(?)

And (probably) more.
